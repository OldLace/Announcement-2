Service Description
Simple Storage Service (S3) Object based file storage; not for application installation. Dropbox uses this service
S3-Glacier Archival storage, can take up to 5 hours to retrieve files; extremely low cost
Elastic Compute Cloud (EC2) Virtual machines in the cloud
Elastic Beanstalk Deploy code to the code; it handles ALL the infrastructure stuff "automagically"
Lambda Serverless; upload your code and your code responses to events, every time you speak to an Amazon Echo or Alexa, it's actually Lambda
Cloudformation allows repeatable environments, you save what you want in a JSON file and it'll go ahead and create it for you, be it one server or a whole stack
